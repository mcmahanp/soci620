Panel data: student test scores over time¶

In this lab, we will be shifting gears to look at panel data. Repeated measurements over the same group of people is a great use case for multilevel models. Because we expect there to be some commonality between the test scores of a particular student over time, we can group indvidual scores at level 1 within students at level 2.

Start by loading a new set of data that is derived from the same Tennessee STAR study we have been using. This time, however, we will look at students across multiple time points: kindergarden through third grade.

As usual, we will subset the data so the model will run faster. The full dataset ha 46,404 observations across 11,601 students. We will look at just 400 of those students.

Random-intercept¶

A simple model of inter-student variation allows the intercept to vary from student to student. Here, we will predict student test scores using just class size:

$$\begin{align} S_{ti} &\sim \mathrm{Norm}(\mu_{ti},\sigma)\\ \mu_{ti} &= \beta_{0i} + \beta_{1}CSize_{ti} \\ \beta_{0i} &= \gamma_{00} + \eta_{0i}\\ \\ \eta_{0i} &\sim \mathrm{Norm}(0,\phi_0)\\ \\ \gamma_{00} &\sim \mathrm{Norm}(500,100)\\ \beta_{1} &\sim \mathrm{Norm}(0,30)\\ \sigma &\sim \mathrm{HalfCauchy}(0,50)\\ \phi_0 &\sim \mathrm{HalfCauchy}(0,50)\\ \end{align}$$

Temporal modelling¶

While the random-intercept model takes into account clustered variation for each student, it does not allow for any kinds of structured change over time, either within or across students. To do this, we expand the model by adding a predictor for grade number. The data has two variables measuring grade: grade is one of gk, g1, g2, or g3, while grade_num is an integer with values zero through three. We will use grade_num to model a linear temporal trend in student scores. Moreover, we will allow this linear trend to vary according to student. Note that we do not have inter-student variation on the coefficient for class sizethere is no need to vary all of yoru slopes randomly if you do not have a reason to do so.

$$\begin{align} S_{ti} &\sim \mathrm{Norm}(\mu_{ti},\sigma)\\ \mu_{ti} &= \beta_{0i} +\beta_{1i}Year_{ti} + \beta_{2}CSize_{ti} \\ \beta_{0i} &= \gamma_{00} + \eta_{0i}\\ \beta_{1i} &= \gamma_{10} + \eta_{1i}\\ \\ [\eta_{0i},\eta_{1i}] &\sim \mathrm{MVNorm}([0,0],\Phi,R)\\ \\ \gamma_{00} &\sim \mathrm{Norm}(500,100)\\ \gamma_{10} &\sim \mathrm{Norm}(0,30)\\ \beta_{2} &\sim \mathrm{Norm}(0,50)\\ \sigma &\sim \mathrm{HalfCauchy}(0,50)\\ \phi_0,\phi_1 &\sim \mathrm{HalfCauchy}(0,50)\\ R &\sim LKJ(2,2) \end{align}$$

Notice that adding this time-dependency actually flips the sign of the coefficient on class size (and would do so 'significantly' if we used the full data set). Also, there is a strong negative correlation between the random effect on grade number and the random intercept. This means that students that have a higher average score will tend to have a lower slope associated with time.

We can compare the WAIC for the two models to see if the fit is better after taking time into account.

It can be useful to plot the predicted temporal trend for individual students. To do this with brms, you can extract the "fixed" and "random" effects.

fixef() returns a matrix with one row for each coefficient and four columns listing the mean, standard error, and percentile interval for each of them.

ranef() returns something more complicated: a list of group-level effects for each of the grouping variables we used. Since we only used one (student_id) it is a list of length 1. Each element of that list (in our case just the one element) is a 3-dimensional array listing the mean, standard error, and percentile range for each student and for each coefficient.
